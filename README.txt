HOW TO RUN USING VS 2022

1. Run SQL file using XAMPP - MySQL
	- XAMPP download link: https://www.apachefriends.org/download.html
	- Install XAMPP (remember to tick the install MySQL option when install)
	- Start Apache and MySQL on XAMPP Control Panel. (Close others MySQL server before click start MySQL)
	- http://localhost/phpmyadmin for access to MySQL server
	- Click Import on the top bar, add Database.sql file and click Import
2. Windows APP
	- Install SAP Crystal Report runtime 32bit and 64bit. link: https://www.tektutorialshub.com/crystal-reports/how-to-download-and-install-crystal-report-runtime/#hownbspto-download (Service Pack 32. VS2022)
	- Run solution file in the App folder
	- Build and run
	- username: admin and pass: admin for admin form
	- username: user and pass: user for user form (main function)
3. Web APP
	- After running the XAMPP - MySQL in step 1
	- In the folder xampp (xampp install location, default location is in C: drive)
	- Open the folder htdocs
	- Copy folder App in the Final_SE_HK1 folder to htdocs folder
	- access http://localhost/web/ for web application.

Warning: Install SAP Crystal Report runtime 32bit and 64bit before build and run, otherwise you can't build or print function not working
